# Hacker News 资料包

**优先用你已有的老账号**——HN 没有头像/品牌位，账号年龄与 karma 就是信用；新号发 Show HN 存活率明显更低。确实没有账号才注册（https://news.ycombinator.com/login 同页 create account）。

| 字段 | 填写内容 |
|---|---|
| 用户名（仅新注册时） | 个人风格即可（如 `spectermyth`），**不要**用产品名当用户名（HN 反感纯营销号） |
| about（个人页可选） | 见下 |
| email 字段 | 可留空或填 ops 邮箱（该字段仅管理员可见；想让人联系需写进 about 正文） |
