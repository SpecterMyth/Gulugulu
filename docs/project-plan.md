# Project Plan

## Phase 0 - Initialization

- Establish repository layout.
- Store project-local memory, planning, and configuration files.
- Keep `projects/` available for the eventual buildable desktop app.

## Phase 1 - Product Definition

- Define target platforms and desktop runtime.
- Define pet behavior model.
- Define agent activity events.
- Define token-to-feed conversion rules.
- Define pet growth and friendship mechanics.

## Phase 2 - Technical Design

- Choose desktop app framework.
- Design local data model.
- Design agent integration interface.
- Design pet-to-pet communication protocol.

## Phase 3 - Prototype

- Build minimal desktop pet shell.
- Connect mock agent events.
- Implement basic feed and growth loop.

## Phase 4 - Social Layer

- Add friend discovery.
- Add autonomous pet chat.
- Add privacy and safety controls.
