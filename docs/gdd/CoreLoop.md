# 核心循环

> ⚠️ **2026-07-21 机制修订**：Token 不再回精力，改为直接折算当前陪伴宠（主宠）的经验；精力只靠「挂机自然恢复（全员生效）+ 敲键盘（仅喂陪伴宠）」两条途径回复；漫游零食已移除，闲逛不回复。下方「精灵等级」「精力恢复」两条已按此重写。

精灵蛋 - 孵化需要时间，孵化完成后变为精灵
精灵 - 点击精灵让它打工，获得钱和经验，点一下赚一次，但会消耗精灵的精力；精力耗尽则精灵趴下充电，恢复到 10% 即站起继续接受点击（2026-07-14 修订）
商店 - 用钱在商店购买不同属性的蛋，可以孵化出不同属性的精灵
精灵等级 - 点击打工和 AI 产出的 Token 都能提升精灵等级（经验的两条来源），但金币仍只从点击打工获得，点一下赚一次；Token 只喂当前陪伴宠（主宠），陪伴宠满级或缺席时整段浪费、绝不溢给其他宠；每日全局最多 2000 次点击获取经验与金币（Token 喂养不占此额度），由玩家在不同宠物间分配（2026-07-14 修订 · 2026-07-21 机制修订）
精力恢复 - 精力只靠两条途径回复、均不产生成长：挂机自然恢复（全员生效）+ 敲键盘（只喂当前陪伴宠，陪伴宠满管或缺席时键次浪费、无溢出转移）；1 阶挂机 10 分钟 / 按键 200 次回满，恢复需求每升一阶 ×5，点击收益同样每阶 ×5；键盘任意按键会播放带该字符的键帽飞向宠物的动画并恢复少量精力。AI 产出的 Token 不再回精力，而是直接折算为当前陪伴宠的经验（全阶统一换算、不随阶数放大），播放 Token 飞向角色的经验增长特效，陪伴宠满级或缺席时 Token 整段浪费；漫游/闲逛只是散步，不再回复任何精力（漫游零食已移除）（2026-07-14 新增 · 2026-07-21 机制修订）
精灵融合 - 将两个满级精灵拖到一起，可以融合成一个高阶蛋，高阶蛋可以孵化出对应这两个精灵的融合体（两个1阶精灵合成2阶，必须两个同阶精灵融合）
孵化屋 - 可以升级，决定了同时可以孵化几个精灵
后院 - 可以升级，决定了最多同时拥有多少个精灵，多出来的要被放生

# 精灵属性
6种属性：一般、火、电、水、草、冰
六种属性各有一个初始精灵，用对应的1级蛋可以孵化出来

# 实现要求
这个游戏所有的操作都在这个透明窗口内产生，不再打开任何网页界面
左键单击当前宠物后，在当前宠物下方显示出完整的游戏菜单，可以在各个功能之间进行切换
当点击对应的功能时，上方原宠物区切换为对应功能的操作区
注意透明窗口的大小要随着显示内容的多少而改变，而不是一个固定的大小

# 风格和题材
使用宠物小精灵、数码宝贝这样的题材
不再局限出现在的鸭子，而是把它当做所有宠物中的一个
