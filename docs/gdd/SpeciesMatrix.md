## 1. 命名与唯一性规范

- **codename**：小写英文 `[a-z]{3,16}`，存档外键**一经定稿永不改名**；不复用 21 个废弃名（guluswan/infernofox/thunderking/tidefrog/mycobeast/glacierpeng/blazeduck/sparkduck/rippleduck/mossduck/frostduck/plasmatanuki/steamander/cinderleaf/thermowolf/stormeel/vinevolt/auroramink/lotusturtle/floeseal/frostbunny）；不以 `aif` 开头（AI 自定义物种前缀保留）。
- **中文名**：2~4 字可爱风（叠字优先），全谱唯一。
- **生物基底全谱唯一**：不复用 6 只一阶的基底（鸭/狐/鼠/鲸/菇/雪怪），也互不重复；概念上回避旧 21 的招牌意象（天鹅/狼/貂 等）。
- **工具全谱唯一**（含与一阶 6 件不重复），职业覆盖：办公 / 服务业 / 工匠 / 科研 / 艺术 / 幻想。
- **配色规则**：主色取该物种气质最贴合的成员元素色（元素色：一般 `#6E6E78`·火 `#E85D3A`·电 `#FFD93B`·水 `#2E7BD6`·草 `#57B84C`·冰 `#8FD8E8`），其余成员元素以点缀件/斑纹呈现；描边统一 `#3B2B1D`，禁用渐变。
- **华丽度 = 元素数**（与阶数无关）：2 元素 = 本体 + 2~3 元素点缀；3 元素 += 1 件背饰/平台；4 元素 += 环绕件（轨道珠/光环）；5 元素 += 多层礼装；6 元素 = 全要素盛装。
- **粒子 = 工具产物**：下表"打工/点击粒子"列须是该物种"打工工具"的直接产物（**≥2 个工具产物 + ≤1 个元素点缀**，用工具/材料固有色）——点击飘散特效读起来就是"手中工具正在产出的东西"。落地实现与 AI 融合物种的同规则见 [SpeciesArtSpec.md](SpeciesArtSpec.md) §7（`TOOL_FX` 目录 / `resolveWorkFx` / `toolFxHints`）。
