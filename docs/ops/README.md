# Gulugulu 全球发行与运营 · Ops 总览

> 目标：把 Gulugulu 作为**免费游戏**推向全球，最大化玩家规模与活跃，带动 Steam 市场交易（收入 = 市场交易抽成）。
> 本目录是发行运营工作的**唯一进度事实源**；技术上架事实源仍在 [plans/steam_trade/](../../plans/steam_trade/README.md)。
> 运营模式：**AI（Claude）执行为主，用户（唯一人类）只做审批、账号级操作与拍板**。任何会话开工前必读本文件 + 所认领文档。

## ⚠️ 定位变更（2026-07-21 用户拍板）

**免费（Free to Play）发行**，主目标 = 拉新 → 活跃 → Steam 市场交易。

- 覆盖 2026-07-16 的 $0.99/¥6 付费拍板（原理由：免费会抬高玩家侧**受限账户**的交易门槛——Steam 账户需累计消费 $5 才能交易/上市场）。
- 应对：核心受众（AI 编程开发者 + Steam 存量玩家）绝大多数早已解除受限；免费带来的漏斗放大对市场流动性是净收益；受限新号仍可正常游玩收集，解限后其库存自然入场。受限门槛反而挡刷号工作室，偏利好。
- **待同步**（用户确认后 🤖 执行）：partner 后台定价改 Free（👤 点保存）、[06-store-page.md](../../plans/steam_trade/06-store-page.md) 定价行、steam_trade README 备注追记。

## ★ 一期最重要的三件事

1. **商店页转化基建**：免费游戏没有价格摩擦，商店页 = 唯一转化引擎。素材文案已就绪 → 补预告片 BGM、描述多语言化（10 语）、后台填写提审，尽快挂出 Coming Soon 蓄势。
2. **开发者圈引爆序列**：本品的钩子（「你的 Claude Code 干活，桌宠吃 token 长大」）天然长在 2026 年最热的 AI 编程人群里。预热 2 周 + 发售 48 小时在 X / Reddit / Show HN / Product Hunt 集中打击，这是零预算下唯一能一次拉满势能的窗口——**发售窗口只有一次**。
3. **自动化运营系统跑起来**：日巡检（评测/讨论/社媒→回复队列）、周内容（GIF 素材+多语言发布）、周报（全指标归因）三条定时管线在发售前空转演练完毕，发售后即无人值守运转。
