# 00 · 战略与定位

> 状态：v1（2026-07-21）。改本文件 = 改战略，须附日期与理由。

## 1. 产品与卖点

**一句话**：陪你写代码的 Steam 桌宠——你的 AI 编程代理（Claude Code / Codex）干活，它吃 token 长大；孵化、融合出由 AI 现场设计的独一无二物种，并作为真实 Steam 资产交易。

**USP（对外叙事的弹药，按强度排序）**：

1. **第一批「AI 编程代理伴侣」**：监听本地 agent 会话日志实时反应（工作/思考/成功/报错/吃 token 进食动画）——精准踩中 2026 年 agent coding 浪潮，开发者一看即懂。
2. **AI 现场造物种**：融合蛋由玩家本机的 Claude/Codex CLI 生成全新物种；创意工坊全球共享，「首个融合出它的人」的形象成为世界线定版。
3. **真 Steam 资产**：掉落/铸造 → Steam 库存 → 交易/市场；63 上架固定物种 + 570 AI 变种槽位，服务器强制的稀有度。
4. **隐私洁癖**：键盘充能只计数不记键（≤250ms 瞬态字形、无 ToUnicode），agent 日志只读本地、不上传；代码公开可审计。→ 这是防「间谍软件」质疑的盾，也是 HN 的加分项。
5. **免费、单机、无服务器**、全 SVG 矢量渲染的 260×320 小窗，不打扰工作。
6. **Meta 反差**（内容支柱 P4）：一个「AI 帮忙造的、AI 负责运营的、靠喂 AI 干活养大的」游戏——运营周报本身就是传播素材。

## 2. 商业模型与北极星

- 收入 = **Steam 市场交易抽成**（游戏方费率以 partner 后台配置为准，行业默认口径 ~10%，Valve 另收 5%）。免费游戏本体 0 收入，一切服务于「更多人玩 → 更多物品流通 → 更多成交」。
- **北极星：周市场成交额**（件数 × 均价）。
- 先行指标链：曝光 → 商店页访问 → 安装（免费一键）→ 激活（连上 agent）→ 留存 → 上架/交易。

| 漏斗层 | 主抓手 | 度量 |
|---|---|---|
| 曝光 | 内容支柱 × 渠道矩阵（§4/§5） | 帖子曝光量、UTM 点击 |
| 访问→安装 | 商店页（预告片/首图/短简介/多语言描述） | partner 流量报表转化率 |
| 安装→激活 | Onboarding 引导「连接 agent」+ 纯点击也能玩 | **成就全局解锁率做代理**（无自建遥测，隐私定位） |
| 留存 | 日常挂机反应 + 收集/融合进度 + 双周更新 | 成就漏斗 + DAU/CCU |
| 交易 | 每日铸造封顶造稀缺 + 行情内容 + 工坊皮肤 | partner 经济报表（成交量/费收入） |

> 遥测原则：**不自建数据上报**（隐私是卖点），漏斗全部用 Steamworks 官方数据 + 成就解锁率反推。

## 8. 风险与合规（全体会话红线）

| 风险 | 对策 |
|---|---|
| 平台规则（刷量/马甲） | 不开小号、不互刷、不激励评测；Reddit 逐 sub 读规则守自我推广比例；X 保持真人节奏（日 1-2 帖）；所有内容以开发者身份透明署名 |
| AI 内容争议 | Steam 内容问卷如实披露（预生成素材 + 玩家侧实时生成）；社媒在平台要求处注明 AI 参与；P4 支柱主动把「AI 参与」变成卖点而非把柄 |
| 「间谍软件？」质疑 | 盾牌前置：隐私契约（键盘只计数、日志只读本地）写进商店页 FAQ + press kit；代码公开可审计；HN 回帖模板常备（kb/faq.md） |
| 受限账户不能交易 | 商店页/FAQ 如实说明「Steam 累计消费 $5 解锁交易」；引导先玩收集；此门槛同时挡刷号，净利好 |
| 刷号倾销市场 | 掉落每日封顶已实装（发行前收紧 R8）；周报盯市场价格曲线，异常时调 generator 权重（不回改 itemdef 编号不变量） |
| 差评集中点 | 未签名 exe 杀软误报（Steam 分发可信链缓解 + FAQ）；WebView2 缺失（installscript 兜底 R2）；「要装 agent 才好玩」预期管理（商店页明说纯点击可玩，agent 是放大器） |
| 账号安全 | 注册/密码/2FA/支付全 👤；🤖 只在已登录会话内操作；异常风控即停并出申诉草稿 |
| 单点审批瓶颈 | 队列设计为每日 ≤10 分钟；审批超时默认不发顺延；发售 48h 提前对齐用户在线时段表 |
