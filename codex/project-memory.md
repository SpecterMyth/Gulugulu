# Project Memory

## Project

Gulugulu is a desktop virtual pet project.

When users work with agent tools such as CodeX or CloudyCode, the pet responds to the user's working activity. Token usage can be converted into pet feed, allowing the pet to grow over time.

Pets can make friends with other users' pets and autonomously chat with each other.

## Repository Rules

- Final buildable application projects belong in `projects/`.
- Codex-related files belong in `codex/`.
- Project plans, memory, configuration, and specifications must stay inside this repository.
- Do not store this project's working files in Codex global directories.

Initialization only. No implementation decisions have been finalized.
