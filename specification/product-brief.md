# Product Brief

## Concept

Gulugulu is a desktop virtual pet that grows alongside the user's agent-assisted work.

## Core Ideas

- Agent activity drives pet reactions.
- Token usage becomes pet feed.
- Long-term work history contributes to pet growth.
- Pets can connect with friends' pets.
- Pets may autonomously talk with each other.

## Initial Non-Goals

- No implementation in the initialization phase.
- No framework choice has been locked in.
- No network architecture has been finalized.
